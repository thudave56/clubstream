# Parent Quick Start (Match Day)

This guide is for parents or volunteers running a stream at the venue.

## What You Need
- A phone with Larix Broadcaster installed
- A stable network connection (venue WiFi or cellular)

## Steps
1. Create the match.
2. Open the match page and follow the "Next steps" checklist.
3. Scan the Larix QR code or open the Larix link on your phone.
4. In Larix, start streaming.
5. Open the scoring page on any device and update the score.
6. (Optional) Add the overlay URL to Larix as a web widget for on-stream scores.

## Troubleshooting
- If the stream does not go live after starting Larix:
  - Confirm you pressed "Start" in Larix.
  - Check your network connection.
  - Ask an admin to verify the stream pool is ready and recover any stuck streams.
